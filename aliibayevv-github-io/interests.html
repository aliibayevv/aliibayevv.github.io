İlgi Alanlarım

İlgi alanlarım birbirini besleyen üç eksende toplanıyor: fintech, web geliştirme ve algoritmik düşünce. Aşağıda bu başlıkları açtım.

Fintech

  • Algoritmik ticaret stratejileri ve geri test (backtesting)
  • Ödeme sistemleri ve dijital cüzdan altyapıları
  • Blok zinciri ve kripto piyasalarının makro davranışı
  • Risk yönetimi ve portföy optimizasyonu
  • Açık bankacılık (Open Banking) API'leri

Web Teknolojileri

  • Modern frontend: HTML5, CSS3, JavaScript (ES6+)
  • Responsive ve erişilebilir arayüz tasarımı
  • Bileşen tabanlı yapılar (React öğrenme aşamasında)
  • Backend ile entegrasyon: REST, JSON, asenkron istekler
  • Performans, lazy-load, optimize edilmiş build

Günlük Çalışma Rutinim

  1. Sabah piyasa açılışını ve önemli finans haberlerini gözden geçirmek
  2. Derse hazırlık: önceki hafta materyallerini tekrar etmek
  3. Küçük bir kod problemi çözmek (LeetCode / kişisel proje parçası)
  4. Bir saat odaklı laboratuvar / proje çalışması
  5. Yeni bir makale veya teknik içerik okumak

Öğrenmek İstediğim Teknolojiler

  • React + TypeScript
    • Next.js ile SSR
    • Bileşen kütüphaneleri (shadcn, Radix)
  • Node.js ve Express
    • REST API tasarımı
    • WebSocket ile gerçek zamanlı veri akışı
  • Python (veri analizi)
    • Pandas, NumPy
    • Finansal verilerle backtesting

Web Teknolojileri Sözlüğü

HTML
Web sayfalarının iskeletini oluşturan işaretleme dili.
CSS
Görsel tasarım ve düzenleme dili; modern sürümünde flexbox ve grid mevcut.
JavaScript
Sayfalara etkileşim ve dinamik davranış kazandıran programlama dili.
Bootstrap
Hazır CSS sınıfları ve bileşenleri sunan frontend kütüphanesi.
API
Uygulamaların birbirleriyle veri alışverişi yapmasını sağlayan arayüz.
